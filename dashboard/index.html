Busca na web (fornecedores)

O servidor consulta a internet via API de busca (Brave ou Google). São só links públicos — confirme preço, frete e estoque em cada site. Cotas e termos de uso são da API que você configurar.

Análise completa (Mercado Livre)

Busca anúncios no MLB pelo termo, cruza com seu custo e aplica as regras de decisão. O servidor precisa de acesso à internet. Se receber 403, defina ML_ACCESS_TOKEN no ambiente onde roda o Python.

Canal de custo: indique se o preço de compra veio de fabricante, atacado, importador, marketplace B2B, etc. Muitos fornecedores continuam listados na internet mesmo sem estoque — o relatório registra o canal e um lembrete para confirmar disponibilidade.
